# Ravindu_SLM
A fine-tuned small language model (SLM) built on Qwen 2.5 1.5B, trained using LoRA via Unsloth on a custom personal dataset. Designed as a personal AI assistant with knowledge of my projects, skills, and Sri Lankan context. Dataset: 2,800 examples (personal + FuseChat). Training: Google Colab T4.
